# <a name="install_top">導入方法</a>

### <a name="download">= ダウンロード<br></a>
ダウンロードはboothから → <a href="https://updraftshop.booth.pm/items/4934082" target ="_blank">仮リンクbooth</a><br>

### <a name="download">= インストール<br></a>
ダウンロードした「UpdraftShop-Core-vX.X.X.unitypackage」と「UpdraftShop-Communi-vX.X.X.unitypackage」を<br>
自身のunityプロジェクトにドラッグ&ドロップ,もしくは左上のメニューから<br>
[Assets] → [Import Package] → [Custom Package...] でインストールします。<br>
<span style="color: red; ">※ 「UpdraftShop-Communi」を使うには「UpdraftShop-Core」のversionが**1.1.0**以上である必要があります。</span>
<br>
<br>
2つとものインストールが完了すると上部のメニューに [UpdraftShop] が表示されます。<br>
